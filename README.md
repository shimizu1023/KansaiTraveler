# StockBoard


## 概要
StockBoard は、株式投資に必要な情報（ウォッチリスト、仮想ポートフォリオ、決算カレンダー）を一元管理できる Web アプリです。  
「シンプルに、必要な情報をすぐ確認」をテーマに、個人投資家の意思決定をサポートします。

## サイトテーマ
株式投資の「銘柄管理・損益可視化・イベント把握」をまとめて行えるダッシュボード型アプリ

## テーマを選んだ理由
- **きっかけ**  
  株取引を行う中で、銘柄メモや決算予定、損益管理がそれぞれ別のツールやサイトに分散しており、判断に時間がかかっていた。  

- **問題点**  
  ・ウォッチ銘柄の株価・出来高・ニュースを複数サイトで確認しなければならない  
  ・ポートフォリオの損益を手作業で計算しており、全体像がつかみにくい  
  ・決算日やイベント情報を直前まで把握しづらい  

- **解決案**  
  ・ウォッチリストで登録銘柄の株価や出来高を一覧表示  
  ・仮想ポートフォリオに数量・価格を入力し、損益やセクター比率を自動で計算  
  ・決算カレンダーで直近イベントを可視化し、銘柄詳細と連動させる  

## ターゲットユーザ
- 短期～中期の売買を行う個人投資家  
- 投資初心者で「情報をまとめて見たい」と考えている人  
- 複数銘柄の進捗や決算予定を効率よく確認したい人  

## 主な利用シーン
- 朝の寄り前にウォッチ銘柄の価格・前日比を一覧でチェック  
- 新規投資前に仮想ポートフォリオで数量・価格を入力し、損益をシミュレーション  
- 週末に決算カレンダーを確認し、来週の投資戦略を立てる  
- 売買後にトレードメモを残して学習や改善に活用  

## 利用方法
1. ユーザー登録／ログイン  
2. 銘柄を検索してウォッチリストへ追加  
3. 保有中または仮想の銘柄をポートフォリオに登録（数量・購入価格を入力）  
4. 「ダッシュボード」でウォッチリスト・損益・決算カレンダーをまとめて確認  
5. 必要に応じてトレードメモを記録し、履歴を振り返る  

## 機能一覧
- ウォッチリスト機能  
  ・銘柄追加／削除  
  ・株価・前日比・出来高の一覧表示  

- 仮想ポートフォリオ機能  
  ・保有数量・価格を入力して評価額／損益を自動計算  
  ・ポートフォリオ全体の資産推移やセクター比率をグラフ化  

- 決算カレンダー機能  
  ・登録銘柄の決算日一覧を表示  
  ・直近1週間の決算予定をピックアップ  

- トレードメモ機能  
  ・売買根拠や振り返りを保存  
  ・過去メモを一覧表示
